Thirteen ways to make it glow

Every direction shares the same Quest Lumen content — the six services, the work, the five-step process — but each is built around its own concept and signature interaction. From a draggable before/after seam to a horizontal world you travel, a cursor-lit room, a hero that assembles as you scroll, and a physics playground you can throw. Hover a preview to interact with it live, or open any direction full screen.

Tip: the last five are 3D / WebGL builds that load model files. To see their previews render, open this page through a local server — e.g. python -m http.server in this folder, then visit localhost:8000. The first eight preview fine opened directly.
