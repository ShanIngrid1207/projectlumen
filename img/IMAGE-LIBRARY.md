# Quest Lumen — verified local image library

All files live in `quest-lumen/img/`. From any `lumen-*.html` (which sit in `quest-lumen/`)
reference them as `src="img/<name>.jpg"`. Every file below was downloaded from Unsplash at
1600px wide and **visually confirmed** — the description matches the pixels.

All are landscape-ish; always display with `object-fit:cover` on a fixed-height box so any
aspect works. No attribution required in-page (Unsplash license), but credit lives here.
